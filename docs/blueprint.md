# **App Name**: CONEX Media

## Core Features:

- Secure Login & Logout: Allows authorized personnel to sign in securely and exit the private network.
- Protected Content Dashboard: Displays a personalized view of exclusive media content and resources accessible only to authorized users.
- Role-Based Access Control (RBAC): Manages which content and features are visible or accessible to users based on their assigned roles.
- AI-Powered Content Curator Tool: Utilizes AI to analyze authorized content and user preferences to suggest relevant documents, articles, or media for consumption within the private network.
- User Profile Management: Allows authorized users to view and update their basic profile details within the system.

## Style Guidelines:

- Primary color: A bold and direct Red (#EB3C47) to signify importance, security, and action.
- Background color: An extremely light, subtly tinted Off-White (#FFF9F9) to provide a clean and expansive canvas.
- Accent color: A soft and supportive Fuchsia (#F1A3CC) used for secondary interactive elements or subtle highlights, providing an analogous contrast.
- Headline and body font: 'Inter', a neutral and highly readable sans-serif, ensuring clear communication and a modern professional appearance.
- Use clear, vector-based icons that convey their meaning efficiently, similar to the prominent lock icon, maintaining a cohesive and modern aesthetic.
- A centralized, minimalist layout with generous negative space to enhance clarity, focus, and ease of navigation for authorized personnel.
- Subtle, professional animations and transitions for button hovers, content loading, and login feedback to enhance user experience without distraction.
